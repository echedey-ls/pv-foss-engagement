ReadTheDocs Statistics for FOSS PV Software#

Tracking engagement for open-source software is challenging, but engagement statistics are helpful in securing development and maintenance funding.

This website summarizes datasets like GitHub star count and ReadTheDocs page view analytics for several open-source software packages in the solar photovoltaics space.

pvlib-python

pvlib-python is a package for simulating the electrical performance of solar photovoltaic power systems.

rdtools

RdTools is an open-source library to support reproducible technical analysis of time series data from photovoltaic energy systems.

pvanalytics

PVAnalytics is a python library that supports analytics for PV systems. It provides functions for quality control, filtering, and feature labeling and other tools supporting the analysis of PV system-level data.

twoaxistracking

twoaxistracking is a python package for simulating two-axis tracking solar collectors, particularly self-shading.

panel-segmentation

Panel-Segmentation is a package for automated metadata extraction of solar PV installations using satellite imagery coupled with computer vision techniques.
